========================================
QUICK START GUIDE
========================================

OPTION 1: Frontend Only (No Database)
--------------------------------------
1. Simply open index.html in any browser
2. Enter city name and get weather
3. History feature won't work without backend

OPTION 2: Full Application (Frontend + Backend + Database)
----------------------------------------------------------

Step 1: Install Prerequisites
- Java JDK 11+
- MySQL 8.0+
- Maven 3.6+

Step 2: Setup Database
- Start MySQL
- Run: mysql -u root -p < database_setup.sql

Step 3: Configure Backend
- Edit application.properties (set MySQL password)
- Create proper folder structure (see above)
- Move Java files to correct packages

Step 4: Build & Run Backend
- mvn clean install
- mvn spring-boot:run
- Backend runs on http://localhost:8080

Step 5: Open Frontend
- Open index.html in browser
- Test weather search
- Click "View History" to test database

========================================
API ENDPOINTS
========================================

POST   /api/weather/save              - Save weather data
GET    /api/weather/history           - Get all search history
GET    /api/weather/city/{cityName}   - Get weather by city
GET    /api/weather/condition/{cond}  - Get weather by condition
GET    /api/weather/{id}              - Get weather by ID
DELETE /api/weather/{id}              - Delete weather record
GET    /api/weather/count             - Get total record count

========================================
CUSTOMIZATION OPTIONS
========================================

Change API Key:
- Edit script.js, line 2
- Get free key from: https://openweathermap.org/api

Change Server Port:
- Edit application.properties: server.port=XXXX
- Edit script.js: backendUrl = 'http://localhost:XXXX/api/weather'

Change Database:
- Edit application.properties
- Update username, password, database name
